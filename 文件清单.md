## 🛠️ 实践环境建议

### 当前已完成章节的实践要求

#### 第1-2章实践（Linux基础）
```bash
最低要求：
├─ 1台Linux虚拟机（CentOS Stream 9）
├─ 4GB内存，20GB存储
├─ 网络连接
└─ SSH访问权限

推荐配置：
├─ 3台Linux虚拟机
├─ 每台8GB内存，50GB存储
├─ 内网互联
└─ 共享存储（NFS）
```

#### 第4章实践（集群架构）
```bash
最低要求：
├─ 5台Linux虚拟机
├─ 每台4GB内存，50GB存储
├─ 千兆网络互联
└─ 共享存储

推荐配置：
├─ 8-10台Linux虚拟机
├─ 每台8GB内存，100GB存储
├─ 万兆网络（可选）
└─ 并行文件系统（Lustre/GPFS）
```

#### 第6章实践（作业调度）
```bash
要求：
├─ 已完成的集群环境
├─ SLURM/PBS调度器安装
├─ MPI环境配置
└─ 测试作业脚本
```
